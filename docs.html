Introduction

Learn how to set up Loop in your server or enhance your everyday use with commands & more.

Server prefix is set to , by default. Use ,prefix set (symbol) to change it for your server.
Getting Started

Loop is an all-in-one Discord bot with moderation, economy, leveling, antinuke, and more. Once invited, use ,setup to begin configuring the moderation system for your server.

Configuration Customization

Customization

Tailor Loop's behavior and appearance to fit your server.

Customizing Loop's avatar

You can change Loop's avatar by using the ,customize avatar command.

Customizing Loop's banner

You can change Loop's banner by using the ,customize banner command.

Customizing Loop's bio

You can set Loop's bio using the ,customize bio command.
